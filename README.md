# mini_blog
<h2>个人练习基于node4.5+express+mongo+bootstrap</h2>
<h2>功能</h2>
<ul>
<li>注册&&登录</li>
<li>修改设置头像</li>
<li>发送添加内容</li>
<li>ajax加载内容</li>
<li>个人主页</li>
<li>所有人列表</li>
<li>更新中...</li>
</ul>
<h2>运行</h2>
<pre>第一步：安装依赖————npm install</pre>
<pre>第二步：运行mongodb————mongod --dbpath c:\mongo</pre>
<pre>第三步：运行服务————node app.js</pre>
<pre>listen in 3000</pre>
